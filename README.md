Основные команды и их функции:

1. Системные команды

/showbizinfo - Включение/выключение отображения информации о бизнесах на экране
/bizgui - Открытие/закрытие графического интерфейса (ImGui окна)
/bupd - Запуск обновления данных о бизнесах (аналог /prodmenu с обработкой)
/pr - Быстрая команда для отправки /prodmenu (без обработки)

2. Команды позиционирования

/setbpos - Активация режима установки позиции отображения текста
/resetpos - Сброс позиции отображения на значения по умолчанию
